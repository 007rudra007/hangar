# Open Wardrobe App

This is the Android application for the Open Wardrobe project.

## Getting Started

To get started with this project, you'll need to have Android Studio installed, along with the Android SDK.

1. Clone this repository.
2. Open the project in Android Studio.
3. Let Gradle sync and download the necessary dependencies.

## Building

To build the application, you can use the following Gradle command:

```
./gradlew assembleDebug
```

## CI/CD

This project is set up with a Continuous Integration and Continuous Deployment (CI/CD) pipeline using GitHub Actions. The pipeline will automatically build and test the project on every push and pull request.
