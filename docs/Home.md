# Hangar Wiki

Welcome to the **Hangar** documentation!

## Project Overview
Hangar is a Virtual Wardrobe application built with web technologies and wrapped for Android using Capacitor.

## Technology Stack
-   **Frontend**: React (Vite)
-   **Styling**: Tailwind CSS
-   **Mobile Bridge**: Capacitor
-   **Platform**: Android
